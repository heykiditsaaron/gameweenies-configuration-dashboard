MVP Goals
- Basic schema-driven UI
- Edit fields for each mod
- Save and deploy configs
- Basic module system
- SQLite database
- Simple RBAC (viewer/editor)
- Basic diff support
- SSH/SFTP deployment
- One config file per mod

Premium Goals
- Advanced custom per-mod UI
- Visual builders (e.g. SkiesShop interface)
- Conditional schema logic
- Rich validation
- Compound fields and layouts
- PostgreSQL database
- Full audit log
- Detailed diff + multi-file support
- Automatic reloads (where mods support it)
- Drift detection
- CI/CD pipelines
- User impersonation and granular permissions
